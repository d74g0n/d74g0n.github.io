    The quickest way to obtain all the addresses used on faucethub is through cryptopia the online wallet and exhange.

    https://www.cryptopia.co.nz/

    An email address confirmation is required. Beyond that, we are not going to ever actually use this for storing coins, we will only use it to generate keys to collect with. Online wallets tend to dissapear. So learn more about CORE wallets before you withdraw from FaucetHub.

    Whatever wallet system you use, you will have to generate deposit keys to attach to faucethub

    Each coin uses a different 'looking' key. Example bitcoin key:

    1LYWGY8UKvDrAbb6J3HfZYHopbrvxyB4N6

    faucethub.io/dashboard/wallets
